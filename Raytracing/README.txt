The program is a simple ray tracer that renders a 3D scene into a 2D canvas, the output image.png is a picture of the rednered canvas.


To run the code you need the following:
  1- Pyhon 3.
  2- Numpy.
  3- Pillow

download Raytrace.py and then go to the terminal and type the following:
  pip install numpy pillow

now the program is ready to run.
